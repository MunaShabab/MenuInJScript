PHP

PHP is a recursive acronym that stands for Hypertext Preprocessor, it originally stood for Personal Home Page. It is a general-purpose scripting language the is used on the server-side web development and is used like javaScript it is used to create dynamic web pages.It is widely used on the web, 78.8% of the server side scripting in 2007.It acts as a filter taking an input stream of content and looking at commands within its delimiters and then creating a dynamic output, mostly HTML, but it can be other things, like images that is used to update the page. PHP is known for its support for a wide range of databases, wich makes creating a dynamic web content from a database fairly simple. Wikipedia You can start to learn PHP at W3Scools. AJAX

AJAX is an acronym that stands for Asynchronous JavaScript and XML,and it describes a set of development techniques used for building websites and web applications. Ajax makes make it possible to update web content asynchronously , meaning updating a web pge without completely reloading it. An example of that would be Google's text suggestions as you starrt typing in the search bar. AJAX is a technique that coordinate the work between the client and the server to produce dynamic and responsive websites using HTML, CSS, JavaScript, and XML. Moreover AJAX focuses only on the parts of a page being updated and ignores the rest resulting in faster website and content loading, it basically works in the background without disturbing the look of the page where no updates are happening. When a user on a website that makes use of AJAX activates an event, JavaScript creates a XMLHttpRequest which is sent to the server, the server processes the request and sends the updates back to the client where it is processed by JavaScript and displayed on the page. You may check out some of those websites to learn more about AJAX, SkillCruch.com, W3Scools, and Wikipedia

jQuery

jQuery is a JavaScript library designed to simplify accessing the HTML DOM and performing different aactions on the pages, so whatever you can do with JavaScript you can do the same using jQuery with much fewer words. It is designed to make it easier to navigate documents and perform common actions, jQuery takes JavaScript code that can many lines long and wraps it up in methods that can be called with a single line of code. According to Wikipedia it is the most widely used JavaScript library on the websites in the world. jQuery is described as a lightweight or write less, do more JavaScript library. It has four principles:

  • Separation of JavaScript and HTML
  • Brevity and clarity
  • Elemination of cross-browser compatibilty
  • Extensibility:New events, elements, and methods can be easily added and then reused as a plugin.

You may check out some of those websites to learn more about jQuery, jQuery.com, W3Scools
